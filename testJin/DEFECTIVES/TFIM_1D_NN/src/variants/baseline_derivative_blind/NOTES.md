# 기준선: 미분맹 분기

## 문제 스토리

1. ReLU 대신 GELU를 사용했기 때문에 Hessian 계산 자체는 가능합니다.
2. 하지만 학습은 여전히 자유에너지 `F` MSE에 거의 지배됩니다.
3. 자유에너지는 매끄럽고 변화폭이 작은 곡선입니다. 반면 상전이 증거는 곡률에 들어 있습니다.
4. 그래서 모델은 작은 `F` MSE를 얻으면서도, `chi` 피크를 흐리게 만들거나 위치를 잘못 잡을 수 있습니다.

## 문제 해결 목표

- 곡률을 학습의 1급 목표로 만들기
- 낮은 온도, `h ~= 1` 근처 derivative 샘플에 집중하기
- global MSE만 보지 말고 피크 위치와 피크 높이까지 평가하기
